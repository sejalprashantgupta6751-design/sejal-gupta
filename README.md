# Sejal Gupta - Portfolio

A modern, clean portfolio website showcasing design work and projects.

## Getting Started

**1. Install dependencies**

```bash
npm install
```

**2. Run development server**

```bash
npm run dev
```

**3. Build for production**

```bash
npm run build
```

## Tech Stack

- **Framework**: Next.js 16
- **UI Library**: Once UI
- **Styling**: React + TypeScript
- **Content**: MDX for blog posts and projects
